# List of commands and maps added

* **File List**
    * :F - Open the file List
    * :E - Toggle the file list

* **Git**
    * gnh - The beginning of the next hunk
    * gph - The end of the last hunk
    * ghs - Stage the hunk
    * ghr - Reset the hunk
    * ghp - Preview the hunk
    * ghb - Blame line
